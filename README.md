# Online Courses Catalog (Test Task)

Тестовое задание: Каталог курсов с фильтрацией и поиском.

## Стек технологий

- HTML5 (Semantic)
- SCSS (BEM, Flexbox, Grid, Adaptive)
- Vanilla JavaScript (ES6+)
- Vite (сборщик)

## Функционал

- Рендер карточек из JS-массива.
- Фильтрация по категориям.
- Живой поиск (с Debounce).
- Подсчет количества курсов в каждой категории в реальном времени.
- Имитация пагинации (кнопка "Load More").
- Адаптивная верстка (Mobile First подход).

## Запуск проекта

1. `npm install`
2. `npm run dev`

## Что можно улучшить (Plans for improvement)

В рамках тестового задания я сфокусировалась на базовых требованиях, но в реальном проекте я бы добавила:

1. **Accessibility (a11y):** Добавить управление с клавиатуры для табов (Tab/Arrow keys) и ARIA-атрибуты.
2. **TypeScript:** Переписать JS на TS для типизации данных курсов.
3. **Empty States:** Улучшить дизайн заглушки, когда ничего не найдено.
4. **Router:** Если бы проект рос, добавить роутинг для открытия страницы конкретного курса.
5. **Тесты:** Добавить Unit-тесты для функций фильтрации.
